Wichtige Codes for Linux

  • 1. cd.
    Cd bedeutet change directory. Es wird verwendet, um das aktuelle Arbeitsverzeichnis zu ändern. Ebenfalls wird es verwendet, um von einem Verzeichnis in ein anderes zu wechseln, auch zum in das übergeordnete Verzeichnis zu wechseln.
  • 2. sudo.
    Sudo bedeutet suposer do. Sudo wird viel verwendet um software zu installieren oder ausserhalb des Home Verzeichnisses des Benutzers zu bearbeiten. Ebenfalls schützt sich Linux so selber sodass eine Benutzer/innen versehentlich Das Dateisystem verändern.
  • 3. ls.
    Ls bedeutet list. Mit dem Befehl ls kann man den Inhalt des jetzigen Verzeichnisses anzeigen lassen.
  • 4. shutdown.
    Shutdown bedeutet ausschalten und wird, wie es schon sagt zum ausschalten eines Rechners verwendet.
    Man kann shutdown now schreiben dass es nicht wie Standardgemäss nach einer Minute herunterfährt, sondern sofort.
  • 5. kill.
    Kill bedeutet benden. Dieser command wird normalerweise verwendet, um einen Prozess anhand seiner Prozess-ID (PID) zu identifizieren und zu beenden.
  • 6. cat.
    Cat steht für concatenate und wird verwendet, um den Inhalt von Dateien anzuzeigen oder Dateien zusammenzufügen. Mit dem Befehl "cat datei.txt" kann der Inhalt der Datei "datei.txt" auf der Konsole angezeigt werden.
  • 7. css.
    css steht für Cascading Style Sheets. Es ist eine Sprache, die verwendet wird, um das Aussehen und die Formatierung eines in HTML geschriebenen Dokuments zu beschreiben. css ermöglicht es Ihnen, das Layout, die Farben, die Schriftarten und andere visuelle Aspekte Ihrer Webseite zu kontrollieren. Es wird verwendet, um optisch ansprechende und reaktionsfähige Websites zu erstellen.
